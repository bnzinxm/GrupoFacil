# GrupoFácil

![Logo do GrupoFácil](./app/assets/logo.png)

**GrupoFácil** é um aplicativo inovador desenvolvido para ajudar professores a gerenciar e organizar grupos de alunos, atividades, e comunicação de maneira simples e eficiente. Com funcionalidades pensadas para facilitar a vida dos educadores, o **GrupoFácil** oferece uma plataforma prática para o controle de turmas e planos de ensino.

## Funcionalidades

- **Criação de Grupos**: Organize turmas e grupos de alunos de forma rápida e intuitiva.
- **Comunicação Direta**: Envie mensagens para grupos de alunos ou alunos individualmente.
- **Planejamento de Aulas**: Crie e compartilhe planos de aula com os alunos.
- **Calendário de Atividades**: Programe eventos, provas e atividades escolares para os alunos.
- **Armazenamento de Materiais**: Compartilhe documentos, apresentações e outros materiais com os alunos.
- **Relatórios de Desempenho**: Acompanhe o progresso dos alunos com relatórios e gráficos.

## Planos de Assinatura

O **GrupoFácil** oferece planos para atender tanto professores individuais quanto escolas e instituições de ensino. Escolha o plano que melhor se adapta às suas necessidades:

- **Plano Básico** (Gratuito)

  - Limite de até 3 grupos de alunos
  - Funcionalidades básicas de comunicação e organização
  - Acesso limitado ao armazenamento de materiais

- **Plano Profissional**

  - Sem limite de grupos de alunos
  - Funcionalidades avançadas de relatórios de desempenho
  - Maior capacidade de armazenamento
  - Acesso ao suporte premium

- **Plano Institucional**
  - Funcionalidade de gerenciamento de múltiplos professores
  - Dashboard personalizado para coordenação pedagógica
  - Suporte prioritário
  - Integração com outras plataformas de gestão educacional

## Como Usar

### 1. **Cadastro**

Primeiro, baixe o aplicativo **GrupoFácil** na Google Play Store ou App Store, ou acesse a versão web.

### 2. **Criação de Conta**

Cadastre-se utilizando seu e-mail ou vincule uma conta de redes sociais para criar sua conta.

### 3. **Configuração Inicial**

Após o cadastro, crie sua primeira turma ou grupo de alunos e comece a adicionar os membros. Defina as configurações básicas do grupo, como nome, descrição e horários.

### 4. **Adicionar Atividades**

Organize suas aulas e atividades, associando materiais de estudo, links e cronogramas diretamente na plataforma.

### 5. **Assinatura de Plano**

Caso precise de mais recursos, acesse a área de configurações para escolher um dos planos pagos.

## Tecnologias Utilizadas

- **Frontend**: React Native (para versão mobile), React (para versão web)
- **Backend**: Node.js, Express
- **Banco de Dados**: PostgreSQL
- **Hospedagem**: AWS, Firebase
- **Outras ferramentas**: Stripe (para pagamentos), Twilio (para comunicação)
